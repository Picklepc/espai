Task

1 Click Copy Prompt to copy the task prompt to your clipboard.
2 Paste it into Claude, ChatGPT, or any AI chat. The AI will respond with file changes and explanations.
3 Apply the file changes the AI described — create or edit the files it specified in your project directory.
4 Paste the AI's full response below, then click Submit to record it and move to review.
The hub will snapshot your files to compute a diff.

Terminal

Full PTY terminal — run PlatformIO builds, Git, serial monitor, Claude Code, or any shell command directly from the browser. Multiple sessions run simultaneously in tabs.

💻

Terminal requires a PTY backend.

Run python ESPAI.py install-deps to install pywinpty (Windows) or ptyprocess (Linux/macOS), then restart the hub.

💻 What can I do here?

  • Build firmware — compile your ESP32 project with PlatformIO
  • Serial monitor — watch live output from a paired device
  • Git — version control your projects and review diffs
  • Run AI agents — Claude or Codex opens here automatically when you run an Agent Bench task
  • General shell — any command you'd run in PowerShell or bash

⚡ Quick start

Click any command to open a shell and run it immediately.

🤖 Agent Bench integration

When you run an Agent Bench task with the Claude Code CLI or Codex CLI adapter, a terminal session opens here automatically. You can watch the agent read files, write code, and report back — and press Ctrl+C to stop it at any time.
